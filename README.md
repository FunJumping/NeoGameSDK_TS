# 项目运行环境
Vue + Typescript + Webpack
# 运行前端
安装依赖
npm install

生成生成环境最小化包
npm run build

# 目录结构
- src是前端代码
- build是webpack构建配置文件
- dist是build后生成的可发布文件
- static是静态css/js/fonts文件，编译时会被复制到dist目录下。
