# 项目运行环境
Typescript

生成生成环境最小化包
cd wallet
tsc

# 目录结构
- wallet是前端sdk代码
- res是静态css/js/fonts文件
- lib是执行引用的js库，wallet工程编译成code.js
- api_panel.html是测试页面，测试调试SDK使用
